# 🎬 Luxury Edition - Quick Start

## 🚀 Start the App

```bash
cd nextask
npm run dev
```

Open browser to `http://localhost:5173`

---

## 🎯 Try These Features NOW!

### 1. New Passkey Format ✅
**Register a new account** to see the new `GF-XXXX-XXXX` format!

- More professional
- Easier to remember
- Better security

### 2. Cinematic Mode ✨ (THE MAIN FEATURE!)
1. Go to **Stories** page
2. Find any story
3. Click the **✨ Sparkles button** (not the play button!)
4. **WOW!** Fullscreen cinematic experience!

**What you'll see:**
- Fullscreen immersive UI
- Animated gradient background
- Floating particles
- Subtitle-style text
- Smooth animations
- "Relax… you're safe ❤️"

### 3. Bedtime Mode 🌙
**In Cinematic Mode:**
1. Move mouse to show controls
2. Click **Moon icon** (🌙) at top right
3. UI dims to deep dark blue
4. Perfect for sleep!

### 4. Sleep Timer ⏰
**In Cinematic Mode:**
1. Move mouse to show controls
2. Click **Clock icon** (⏰) at top right
3. Choose: 10, 20, 30, or 60 minutes
4. Audio fades out and stops automatically

### 5. ElevenLabs Premium Voice 🎤 (Optional)
1. Get free API key: https://elevenlabs.io
2. In Stories page, click **Settings** (⚙️)
3. Paste your API key
4. Enter Cinematic Mode
5. Enjoy premium Telugu narration!

## 🎮 Controls

### In Cinematic Mode
- **Move mouse** → Show controls
- **X button** → Exit to normal view
- **Play/Pause** → Control playback
- **Volume sliders** → Adjust voice & music
- **Moon icon** → Toggle bedtime mode
- **Clock icon** → Set sleep timer

### In Normal Mode
- **Play button** (▶️) → Background playback
- **Sparkles button** (✨) → Enter cinematic mode
- **Heart icon** → Add to favorites

---

## 🎵 Audio Features

### Smart Audio Ducking
Music automatically lowers when narration plays!
- Music: 30% volume normally
- Music: 15% volume during speech
- Smooth transitions

### Volume Controls
- **Voice**: 0-100% (narration)
- **Music**: 0-100% (background)
- Adjust independently

## 🐛 Quick Fixes

**No sound?**
→ Check volume sliders, browser permissions

**Animations laggy?**
→ Close other tabs, use Chrome/Edge

**Passkey not working?**
→ Format must be `GF-XXXX-XXXX` (uppercase)

**ElevenLabs not working?**
→ Check API key, verify credits
